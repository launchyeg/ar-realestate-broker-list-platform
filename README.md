# AR Realestate Broker List Platform

A broker listing web system built with Next.js, Tailwind CSS, and Supabase, designed to help real estate brokers showcase properties and projects, manage listings, and track client inquiries through an integrated dashboard.

### go to [website](https://ar-realestate-redsea.com)

## System Overview

The system is divided into two main parts:

### 🔹 Client Side (User Interface)

A complete front-facing website designed for real estate brokers, including:

- Home Page
- About Us
- Destinations
  - All Destinations
  - Destination Detail
- Projects
  - All Projects
  - Project Detail
- Properties
  - Property Detail
- Contact Us
- Privacy Policy

**Features:**

- Browse properties by destination and project
- Send booking and viewing requests via contact form
- WhatsApp direct contact integration
- Mobile responsive with sidebar navigation
- SEO optimized with sitemap and structured data
- JSON-driven content — update units.json to add new properties instantly
- Rebrandable in minutes via siteConfig.js
